> **US12 不是百科，也不是课程，更不是主界面**

> 👉 **随时可跳转的「认知底座 / 世界模型速览页」**

## 三、左侧：知识结构树（Ontology Tree）

### 1️⃣ 层级设计（V1 固定三层）

```
Level 1：赛道
Level 2：技术板块
Level 3：子技术 / 技术路线
```

**示例（量子）**

```
量子科技
├─ 量子计算
│  ├ 超导量子计算
│  ├ 离子阱
│  ├ 中性原子
│  └ 光量子
├─ 量子通信
│  ├ 量子密钥分发
│  └ 量子网络
└─ 量子传感 / 测量
   ├ 量子磁测
   └ 原子钟
```

### 2️⃣ 交互规则（很重要）

* 默认 **折叠**
* 只支持 **展开 / 收起 / 点击**
* 不支持编辑（V1）

> 这是**认知结构**，不是用户笔记
