# 市场参与者发现方法论

市场参与者发现必须通过**系统性方法驱动**，而不是依赖预先写好的名单。名单永远滞后于现实，方法论才能应对任何领域、任何时间窗口的研究需求。

---

## 一、价值链拆解（覆盖每个产业环节）

任何技术领域都有层叠的价值链结构。做市场分析前，先识别该赛道的价值链层次，再逐层搜索参与者，确保不遗漏任何环节：

| 通用层次（按研究领域调整） | 典型问题 | 搜索切入方向 |
|------------------------|---------|-----------|
| 基础材料 / 核心器件 | 谁供应关键原材料、元器件或基础工艺？ | "[领域] 材料 器件 供应商 制造" |
| 核心硬件 / 整机系统 | 谁造完整设备或系统？ | "[领域] 硬件 系统 整机 设备" |
| 软件栈 / 算法工具 | 谁做核心软件平台或算法框架？ | "[领域] 软件 算法 框架 SDK" |
| 平台 / 云服务 | 谁提供服务化或 API 形式的接入？ | "[领域] 云平台 SaaS API 服务" |
| 垂直行业应用 | 谁把通用能力落地到具体场景？ | "[领域] 应用 解决方案 行业 场景" |
| 安全 / 合规 | 谁提供配套安全或合规产品？ | "[领域] 安全 加密 合规 认证" |

对每一层分别执行检索。**遇到某层无结果不要跳过——空白本身是投资信号**，三种情况含义完全不同：
- 国内尚无玩家 → 潜在进入机会，或进入壁垒极高
- 被大机构垄断 → 需评估竞争可行性
- 技术尚不成熟 → 判断时间窗口

---

## 二、早期标的发现（Deal Sourcing 核心）

头部公司的公开信息已被市场充分定价，真正的 Alpha 在市场尚未追踪到的早期公司。

**信号层次（按可信度与可操作性排序）：**

| 信号强度 | 信号类型 | 发现方式 |
|---------|---------|---------|
| 🔴 强 | 有可核实融资记录，且创始团队背景可查 | `"[领域] 天使轮 种子轮 pre-A 融资"`, topic="finance" |
| 🔴 强 | 高校 / 科研院所正式孵化，有公开公告 | `"[领域] 孵化 高校 实验室 成立"` |
| 🟡 中 | 核心科学家/工程师公开宣布创业，有媒体报道但融资未披露 | `"[领域] 创业 博士 创始人 离职"` |
| 🟡 中 | 出现在行业媒体但来源单一，尚不可交叉验证 | 标注"⚪ 弱信号，待验证" |
| ⚪ 弱 | 专利聚集或高活跃论文团队，尚无公司注册记录 | 联合 paper-analysis 输出，补搜 `"[团队/导师名] 创业 公司"` |

**发现新公司后，必须补充核查这五项（有则填，无则标注缺失）：**
1. 首次公开时间（成立时间 vs 首次媒体曝光时间）
2. 创始团队来源（学术机构 / 大厂离职 / 海归背景）
3. 技术路线
4. 资方情况（已知投资人、融资规模、战略/财务性质）
5. 机构背景（是否为高校正式孵化项目）

> ⚪ 弱信号只作为"线索提示"，不作为报告主要结论，标注"[待验证]"后列入附录供后续跟进。

---

## 三、完整性验证

初步发现完成后，逐项确认：

- [ ] **价值链覆盖**：每个环节都执行了搜索，空白环节已说明原因（空白 ≠ 可跳过）
- [ ] **早期探测已执行**：第二节的信号类型搜索已运行，即使无结果也须在报告中注明
- [ ] **多源交叉验证**：每条融资数据都有两个以上来源（DB + Web）
- [ ] **时间戳完整**：每家公司都有首次出现/成立时间，每条融资都精确到年月（YYYY-MM）
- [ ] **大型机构已核查**：赛道相关的大型机构当前战略状态已通过 Web 搜索实时确认

---

## 四、大型机构战略核查（每次报告前必查）

大型企业在特定赛道的进退对生态格局影响极大，且变化速度难以从已有认知预判。**必须每次通过 Web 实时核查，不可依赖训练知识或上次报告的结论。**

核查关键词模板：
- `"[公司名] [赛道关键词] 战略 布局 投资 合作"` — 了解当前参与程度
- `"[公司名] [赛道关键词] 裁撤 退出 关停 捐赠"` — 确认是否已退出
- `"[公司名] [赛道关键词] 新进展 产品 融资"` — 是否有新动作

**退出行为本身是投资信号**：大机构撤退、资产转让、团队解散，往往是初创公司切入的窗口期，需在报告"前沿信号"章单独标出。
