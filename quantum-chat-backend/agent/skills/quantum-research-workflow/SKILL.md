# quantum-research-workflow

## When to Use
- 用户粘贴了一篇论文摘要，需要深度分析
- 用户提到某个融资事件，需要判断产业化信号强度
- 用户询问某条技术路线的最新进展
- 用户需要梳理某个公司/机构的研究地图

## Steps

### 1. 信号分类
首先判断输入属于哪类信号：
- `paper`：学术论文 / arxiv 预印本
- `funding`：融资事件 / 估值变化
- `tech`：技术发布 / 产品里程碑
- `policy`：政府政策 / 标准制定

### 2. 技术成熟度评估（TRL 框架）
针对 paper/tech 类信号，评估技术成熟度等级：
- TRL 1-3：基础理论，距商业化 10 年以上
- TRL 4-6：实验室验证，距商业化 5-10 年
- TRL 7-9：工程化 / 商业化，存在近期投资价值

### 3. 竞争格局分析
- 全球谁在做同类方向（公司/机构/研究者）
- 中国玩家现状（有无自主技术、融资状态）
- 与该信号最相关的 2-3 个投资标的

### 4. 关键人识别
- 论文通讯作者背景（学术 or 产业化意向）
- 是否有创业记录或专利转化记录
- 值得追踪的研究者列表

### 5. 输出格式
```
【信号类型】...
【技术成熟度】TRL X — 说明
【核心结论】1-2 句话
【竞争格局】...
【关键人】...
【投资线索】（如有）...
```
