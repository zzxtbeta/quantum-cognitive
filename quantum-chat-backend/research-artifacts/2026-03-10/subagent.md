# 量子计算云平台及操作系统领域人才情报分析报告

## 1. 领先量子云平台背后的核心人才与团队

中国领先的量子云平台主要由顶尖科研机构与头部初创企业主导，形成了“科学家领衔+工程团队支撑”的格局。

### 核心平台及负责人
| 云平台名称 | 运营/研发机构 | 首席科学家/核心负责人 | 关键技术背景 |
| :--- | :--- | :--- | :--- |
| **Quafu (量子算力平台)** | 北京量子信息科学研究院 (BAQIS) | **于海峰** (硬件负责人)、**胡孟军** (软件负责人) | 于海峰负责超导硬件，胡孟军主导量子编译软件 QSteed 研发。 |
| **本源量子云** | 本源量子 (中科大系) | **郭光灿** (首席科学家)、**郭国平** (创始人/首席科学家) | 依托中科大郭光灿院士团队，国内首个上线量子云平台的团队。 |
| **天衍量子云** | 中电信量子 (中国电信) | **张辉** (CTO) | 整合中国电信算力资源与本源量子技术，侧重产业化应用。 |
| **量旋云平台** | 量旋科技 | **项金根** (CEO) | 侧重核磁共振量子计算及超导量子云服务。 |
| **百度量子平台** | 百度 (已捐赠给北量院) | **段润尧** (原负责人) | 曾主导量桨 (Paddle Quantum) 及量子脉冲计算服务。 |

## 2. 重点机构在操作系统及软件研发上的布局

### 北京量子信息科学研究院 (BAQIS)
*   **布局重点**：全栈量子计算系统，特别是量子操作系统与云平台的集成。
*   **核心团队**：量子操作系统软件团队。
*   **关键人物**：**胡孟军**（副研究员）。他主导研发了量子编译系统软件 **QSteed**，并已部署于 Quafu 平台。其研究涵盖量子编译、应用软件及量子 AI。
*   **硬件支撑**：**于海峰**、**燕飞**、**金贻荣**等超导量子计算团队负责人，为软件层提供底层硬件接口支持。

### 中国科学技术大学 (USTC) / 本源量子
*   **布局重点**：国内最早提出“量子操作系统”概念，研发了 **OriginQ Origin Pilot**。
*   **核心团队**：中科院量子信息重点实验室。
*   **关键人物**：**郭国平**、**张辉**、**陈昭昀**。团队在量子芯片测控、量子语言 (QRunes) 及编译器方面有深厚积累。
*   **特点**：产学研结合极紧密，本源量子作为承接主体，实现了从实验室原型到商业云平台的转化。

### 清华大学
*   **布局重点**：量子算法、量子纠错及量子网络软件。
*   **核心团队**：交叉信息研究院 (IIIS)。
*   **关键人物**：**段路明**（离子阱/超导硬件）、**龙桂鲁**（量子算法/通信）。
*   **特点**：侧重于底层算法优化与新型量子架构的软件模拟。

## 3. 人才流向及团队实力对比

### 人才流向趋势
1.  **学术界向新型研发机构流动**：大量中科大、清华、物理所的博士/博士后流向 **北京量子院 (BAQIS)**。例如，胡孟军曾任职于华为，后加入北量院；燕飞从南科大加入北量院。
2.  **大厂人才回流科研/创业**：随着互联网大厂（如百度、阿里巴巴）调整量子实验室布局，核心人才流向北量院或初创企业。
3.  **校企联合培养**：如浙江大学与杭州逻辑比特科技有限公司的合作，学生在校期间即参与企业工程研发（如吴耀祖）。

### 团队实力对比
| 机构 | 软件/操作系统实力 | 硬件支撑能力 | 产业化程度 |

| **北京量子院** | 强（Quafu 平台活跃度高） | 极强（多路线并行，超导比特数领先） | 中（侧重公共算力服务） |
| **中科大** | 极强（理论与实验双领先） | 极强（九章/祖冲之号） | 高（通过本源/国盾转化） |
| **量旋科技** | 中（侧重教学与小型化） | 中（核磁共振/超导） | 高（海外市场拓展较好） |

# 量子计算云平台及操作系统商业化调研报告（2025-2026）

## 1. 商业模式分析：从“算力民主化”到“量超智融合”

全球及中国量子云平台的商业模式已从早期的免费科研测试转向多元化的商业交付，核心逻辑在于降低昂贵硬件的使用门槛。

| 模式类型 | 具体形式 | 典型代表 | 商业逻辑 |
| :--- | :--- | :--- | :--- |
| **按需付费 (Pay-as-you-go)** | 按任务提交次数、量子比特使用时长或Shot次数计费 | AWS Braket, IBM, 玻色量子 | 适合初创企业和开发者，门槛最低。 |
| **订阅制 (Subscription)** | 分级会员制，提供固定额度或优先排队权 | IBM Quantum Network, Microsoft Azure | 锁定长期客户，提供稳定的现金流。 |
| **科研/战略合作** | 联合研发、定制化算法开发、私有化部署 | 本源量子, Qilimanjaro, IonQ | 针对金融、制药等高敏感行业，提供全栈解决方案。 |
| **聚合器模式** | 统一API接入多种第三方硬件资源 | AWS Braket, Microsoft Azure Quantum | 平台方不一定自研硬件，通过生态集成赚取服务费。 |

## 2. 中国主要玩家及核心产品

中国量子计算赛道已形成“国家队+科技巨头+初创领军”的格局，实现了从芯片到操作系统的全栈自主化。

### 2.1 领军企业与核心产品
*   **本源量子 (Origin Quantum)**：
    *   **核心产品**：**“本源司南” (Origin Pilot)** 量子计算机操作系统（全球首个开放下载的本地部署系统 [1][6]）、“本源悟空”系列超导量子计算机。
    *   **地位**：国内首家实现量子计算机整机交付的企业，操作系统兼容超导、离子阱、中性原子等多种路线 [9]。
*   **玻色量子 (BosonQ)**：
    *   **核心产品**：**“量子云枢”** 编程套件、1000计算量子比特相干光量子计算机 [11][12]。
    *   **特色**：专注于专用量子计算（相干光路线），其云服务已上线阿里云市场，实现“量智融合” [11]。
*   **国盾量子 (QuantumCTEK)**：
    *   **核心产品**：接入“祖冲之号”同款176比特超导量子计算机的云平台 [10]。
    *   **特色**：强调安全合规，集成QKD（量子密钥分发）技术，通过等保三级认证 [10]。
*   **量旋科技 (SpinQ)**：
    *   **核心产品**：量旋“双子座”系列桌面级核磁共振量子计算机、量旋云平台。
*   **华翊量子 (HYQ)**：
    *   **核心产品**：离子阱量子计算原型机及云平台。
*   **科技巨头 (百度、阿里、腾讯)**：
    *   **百度**：发布“量桨” (Paddle Quantum) 平台，后将量子实验室捐赠给北京量子信息科学研究院。
    *   **阿里**：阿里云与玻色量子深度合作，上线国产量子计算核心工具套件 [11]。
    *   **腾讯**：腾讯量子实验室 (TQL) 专注于量子化学模拟及药物研发算法。
